Getting Started

Requirements

Before using AngularForms, add the following to your project:

  • JQuery 1.10.2+
  • JQueryUI 1.10.3+
  • AngularJS 1.2.6+
  • FontAwesome 4.0.3+
  • Twitter Bootstrap style sheet 3.0.3+

The above can be installed using Bower:

bower install angular
bower install bootstrap
bower install jquery
bower install jqueryui
bower install components-font-awesome

Download AngularForms

There's more than one way to acquire AngularForms and incorporate it into your project. Choose the method that works best for you:

Just the javascript, please

The quickest way to get started is to download the compiled javascript.

Clone or fork via Github

Cloning or forking the github repo gives you access to a full sample application. Plus, you can contribute!

Install via Bower

Add AngularForms to your project with bower:

bower install angular-forms.js

Sample Template

Below is a very simple html page showing how to incorporate AngularForms. It assumes bower was used to install the required components. Use it as an example and customize it to fit your environment.

<!DOCTYPE html>
<html lang="en" ng-app="myApp">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AngularForms | angular-forms.js. Web forms made simple.</title>
    <link rel="stylesheet" href="/bower_components/bootstrap/dist/css/bootstrap.min.css" />
    <link rel="stylesheet" href="/bower_components/components-font-awesome/css/font-awesome.min.css" />
    <link rel="stylesheet" href="/bower_components/jqueryui/themes/overcast/jquery-ui.min.css" />
    <link rel="stylesheet" href="/bower_components/angular-forms.js/angular-forms.css" />
    <link rel="stylesheet" href="/css/myStyles.css" />
</head>
<body>
    <div ng-controller="yourController">
        <p>Site content appears here.</p>

        <div id="insert-form-here"></div>
    </div>
    <script src="/bower_components/jquery/jquery.min.js"></script>
    <script src="/bower_components/jqueryui/ui/minified/jquery-ui.min.js"></script>
    <script src="/bower_components/angular/angular.min.js"></script>
    <script src="/js/myApp.js"></script>
    <script src="/bower_components/angular-forms.js/angular-forms.min.js"></script>
</body>
</html>

How does it work?

In sampleForm.js angular.module is used to created object named SampleForm. SampleForm is the JSON representation of the HTML we want AngularForms to generate. SampleForm containes form properties (name and horizontal) an object called fields and an object called buttons.

The fields object contains an object for each input field on the final form. For example, the First name input field is reprented in the JSON as fields.first_name. The fields.first_name object contains key:value pairs representing the attributes the input field will receive. You'll see that buttons work exactly the same way.

In myApp.js the Angular application is defined along with a controller called myController. Within the controller the JSON object representing the form is passed to AngularForms(), which turns it into HTML. It also adds CSS classes and Angular directives, providing the form with styling and validtion. The form is generated according to the key:value properties in each field object. The result is compiled and injected into the '#exampleForm' DOM element.

Note the CSS stylesheet included in this example is not currently included in the build. If you're downlading the zip file to install AngularForms, you may want to copy it into your app and adjust as needed. If you're using the source from Github or via Bower, see the stylesheet in the sample app.

myApp.js 
'use strict';

var myApp = angular.module('sampleApp', ['AngularFormsModule', 'SampleFormDefinition']);

myApp.controller('myController', ['$scope', 'AngularForms', 'SampleForm', 
    function($scope, AngularForms, SampleForm) {
    
    var form = AngularForms({ scope: $scope, targetId: 'basicForm', form: SampleForm });       
    form.inject();

    $scope.sources = [
            { id: 'google', label: 'Google' },
            { id: 'yahoo', label: 'Yahoo!' },
            { id: 'bing', label: 'Bing' },
            { id: 'facebook', label: 'Facebook' },
            { id: 'word', label: 'A friend told me' },
            { id: 'other', label: 'Other' }
            ];
    
    }]);
sampleForm.js 
'use strict';

angular.module('SampleFormDefinition', [])
    .value(
    'SampleForm', {
        name: 'SampleForm',                    // Required. Used to set <form> id attribute
        horizontal: false,                     // Set to true for Bootstrap horizontal form layout 
        
        fields: {
            first_name: {
                label: 'First Name',
                srOnly: true,
                type: 'text',
                placeholder: 'First name',
                required: true,
                autocomplete: false,
                helpText: 'Provide your first name.'
                },
            last_name: {
                label: 'Last Name',
                type: 'text',
                srOnly: true,
                placeholder: 'Last name',
                required: true, 
                autocomplete: false,
                helpText: 'Provide your last name.'
                },
            email_address: {
                label: 'Email',
                type: 'email',
                srOnly: true,
                placeholder: 'Email address',
                required: true,
                autocomplete: false,
                helpText: 'Your primary email address.'
                },
            username: {
                label: 'Username',
                srOnly: true,
                placeholder: 'Username',
                type: 'text', 
                length: 30,
                required: true,
                autocomplete: false,
                helpText: 'Choose the name you will be known as.'
                },
            password: {
                label: 'Password',
                srOnly: true,
                placeholder: 'Password',
                helpText: 'Create a password. Make it strong!',
                type: 'password', 
                length: 16, 
                autocomplete: false,
                required: true
                },
            referral_source: {
                label: 'Referral source', 
                srOnly: true,
                placeholder: 'Select a referral source',
                type: 'select',
                required: true,
                helpText: 'How did you hear about us?',
                optionArray: 'sources',
                ngChange: "sourceSelected()"
                },
            other_source: {
                label: 'Other',
                srOnly: true,
                placeholder: 'Other source',
                type: 'text',
                ngShow: "referral_source == 'other'",
                ngRequired: "referral_source == 'other'",
                helpText: 'What is the Other source?'
                }
            },

        buttons: {
            save: {
                label: "Save",
                icon: "fa-refresh",
                ngClick: "save()",
                'class': 'btn-primary btn-sm'
                },
            reset: {
                label: "Reset",
                icon: "fa-minus-circle",
                ngClick: "reset()",
                'class': 'btn-default btn-sm'
                }
            }
        });

How do I learn more?

The above example will get you started building a simple form. Visit the reference guide to see all the available form controls and options.

For more examples, download the source code from Github and take a look at the sample app. It can be run locally using the included node web server:

cd ~/angular-forms.js
node ./scripts/web-server.js
