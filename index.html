AngularForms

Create web forms as reusable JSON objects that can be injected anywhere into your app, complete with Twitter Bootstrap styling and AngularJS data bindings and validation.

Requirements

Before using AngularForms, add the following to your project:

  • JQuery 1.10.2+
  • AngularJS 1.2.6+
  • FontAwesome 4.0.3+
  • Twitter Bootstrap style sheet 3.0.3+

The above can be installed using Bower:

bower install angular
bower install bootstrap
bower install jquery
bower install components-font-awesome

Download AngularForms

Sample Template

Below is a very simple html page showing how to incorporate AngularForms. It assumes bower was used to install the required components. Use it as an example and customize it to fit your environment.

<!doctype html>
<html lang="en" ng-app="myApp">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AngularForms | angular-forms.js. Web forms made simple.</title>
    <link rel="stylesheet" href="/bower_components/bootstrap/dist/css/bootstrap.min.css" />
    <link rel="stylesheet" href="/bower_components/components-font-awesome/css/font-awesome.min.css" />
    <link rel="stylesheet" href="/css/myStyles.css" />
</head>
<body>
    <div ng-controller="yourController">
        <p>Site content appears here.</p>

        <div id="insert-form-here"></div>
    </div>
    <script src="/bower_components/jquery/jquery.min.js"></script>
    <script src="/bower_components/angular/angular.min.js"></script>
    <script src="/js/myApp.js"></script>
    <script src="/bower_components/angular-forms.js/angular-forms.min.js"></script>
</body>
</html>

Getting Started

In the following example sampleForm.js contains a JSON object representing a form. In myApp.js the Angular application is defined along with the controller. In the controller a call to AngularForms() turns the JSON into HTML, adding in CSS classes and Angular directives.

The form is generated based on the properties and attributes found in the JSON object. The result is compiled and injected into the '#exampleFrom' DOM element. Click the button to see it in action.

Note the CSS we add to style a few items that come standard with a generated form. This style sheet is not currently included in the build, so you might want to copy it into your app and adjust as needed.

HTML 
<!DOCTYPE html>
<html lang="en" ng-app="myApp">
    <head>
        <meta charset="utf-8">
        <title>My Sample App</title>
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link href="/bower_components/bootstrap/dist/css/bootstrap.min.css" rel="stylesheet">
        <link href="/bower_components/components-font-awesome/css/font-awesome.min.css" rel="stylesheet">
        <link href="/css/styles.css" rel="stylesheet">
    </head>
    <body>
        
        <div class="container">
            <div class="row">
                <div class="col-lg-12" ng-controller="myController">
                    <h4>Example Form</h4>
                    <div class="well">
                        <div id="exampleForm"></div>
                    </div>
                </div>
            </div>
        </div>

        <script src="/bower_components/jquery/jquery.min.js"></script>
        <script src="/bower_components/angular/angular.min.js"></script>
            
        <!-- angular app files -->
        <script src="/js/myApp.js"></script>
        <script src="/js/sampleForm.js"></script>
        <script src="/bower_components/angular-forms.js/angular-forms.min.js"></script>
    </body>
</html>

myApp.js 
'use strict';

var myApp = angular.module('sampleApp', ['AngularFormsModule', 'SampleFormDefinition']);

myApp.controller('myController', ['$scope', 'AngularForms', 'SampleForm', 
    function($scope, AngularForms, SampleForm) {
    
    var form = AngularForms({ scope: $scope, targetId: 'basicForm', form: SampleForm });       
    form.inject();
    
    }]);
sampleForm.js 
'use strict';

angular.module('SampleFormDefinition', [])
    .value(
    'SampleForm', {
        name: 'SampleForm',                    // Required. Used to set 
 id attribute
        horizontal: false,                     // Set to true for Bootstrap horizontal form layout 
        
        fields: {
            first_name: {
                label: 'First Name',
                srOnly: true,
                type: 'text',
                placeholder: 'First name',
                required: true,
                autocomplete: false,
                helpText: 'Provide your first name.'
                },
            last_name: {
                label: 'Last Name',
                type: 'text',
                srOnly: true,
                placeholder: 'Last name',
                required: true, 
                autocomplete: false,
                helpText: 'Provide your last name.'
                },
            email_address: {
                label: 'Email',
                type: 'email',
                srOnly: true,
                placeholder: 'Email address',
                required: true,
                autocomplete: false,
                helpText: 'Your primary email address.'
                },
            username: {
                label: 'Username',
                srOnly: true,
                placeholder: 'Username',
                type: 'text', 
                length: 30,
                required: true,
                autocomplete: false,
                helpText: 'Choose the name you will be known as.'
                },
            password: {
                label: 'Password',
                srOnly: true,
                placeholder: 'Password',
                helpText: 'Create a password. Make it strong!',
                type: 'password', 
                length: 16, 
                autocomplete: false,
                required: true,
                confirm: true      //generate a confirmation password field. associate the two and add validation directive 
                },
            referral_source: {
                label: 'Referral source', 
                srOnly: true,
                placeholder: 'Select a referral source',
                type: 'select',
                required: true,
                helpText: 'How did you hear about us?',
                optionArray: 'sources',
                ngChange: "sourceSelected()"
                },
            other_source: {
                label: 'Other',
                srOnly: true,
                placeholder: 'Other source',
                type: 'text',
                ngShow: "referral_source == 'other'",
                ngRequired: "referral_source == 'other'",
                helpText: 'What is the Other source?'
                }
            },

        buttons: {
            save: {
                label: "Save",
                icon: "fa-check",
                ngClick: "save()",
                'class': 'btn-primary btn-sm'
                },
            reset: {
                label: "Reset",
                icon: "fa-minus-circle",
                ngClick: "reset()",
                'class': 'btn-default btn-sm'
                }
            }
        });
