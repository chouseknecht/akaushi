With AngularForms your forms become reusable JSON objects that can be injected anywhere into your app, complete with Twitter Bootstrap styling, AngularJS bindings and validation. Let AngularForms handle the forms while you focus on building your app.

Adding AngularForms to your app

AngularForms requires JQuery, AngularJS, FontAwesome, and the Twitter Bootstrap style sheet. Once you have the requirements installed, download the project and add a script tag to your project for angular-forms.min.js.

An index.html would look like the following: 

<!doctype html>
<html lang="en" ng-app="sampleApp">
<head>
    <meta charset="utf-8">
<head>
    <link rel="stylesheet" href="/bootstrap/dist/css/bootstrap.min.css" />
    <link rel="stylesheet" href="/font-awesome/css/font-awesome.min.css" />
    ...
</head>
<body>
    <script src="/jquery/jquery.min.js"></script>
    <script src="/angular/angular.min.js"></script>
    ...
    <script src="/angular-forms.min.js"></script>
</body>
</html>

AngularForms can also be installed into a project using bower:

bower install angular-forms.js

By default AngularForms will be installed in the bower_components directory.
